Place demonstration videos for each letter here.

File naming:
- Uppercase letter, extension mp4 or webm
- Example: A.mp4 (preferred) or A.webm

Paths used by the app:
- /public/videos/letters/A.mp4 (first choice)
- /public/videos/letters/A.webm (fallback)

Behavior:
- The Learn page modal will attempt to load the video for the selected letter.
- If playback fails or the file is missing, a friendly message is shown.
- On successful video end, the letter is auto-marked as completed.
